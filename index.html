H1 até H6

A tag h é utilizada para definição do título e sub-títulos da página. Ela vai do h1 até o h6, onde h1 é o título e os demais são os sub-títulos de acordo com o nível.

(início)

P

A tag p representa um parágrafo. São utilizados, geralmente, com blocos de texto, separando dos elementos próximos por espaços em branco verticais.

(início)

Mark

A tag mark é usada para marcar ou destacar o texto de interesse ou relevância especial em um documento HTML. Por padrão, o texto marcado com essa tag fica com a cor de fundo amarela.

(início)

Small

A tag mark é usada para diminuir a fonte em um intevalo definido de um texto.

(início)

I

A tag i é usada para deixar o texto em itálico (letras deitadas).

(início)

U

A tag u é usada para sublinhar o texto delimitado por ela.

(início)

Strong

A tag strong é usada para colocar o texto delimitado por ela em negrito.

(início)

Ol

A tag ol define uma lista ordenada.

(início)

Ul

A tag ul define uma lista não ordenada.

(início)

Li

A tag li define os ítems de uma lista.

(início)

A

A tag a cria um link, ou seja, uma ligação para uma outra página web ou arquivos. Tem como principal atributo o href, onde é definido o destino do link.

(início)

Hr

A tag hr é usada para quebrar linha, exibindo uma linha horizontal separando os conteúdos da página.

(início)

Sub

A tag sub é usado para escrever um texto com letras menores e alinhamento abaixo do restante do texto.

(início)

Sup

A tag sup é usado para escrever um texto com letras menores e alinhamento acima do restante do texto.

(início)

Blockquote

A tag blockquote é usado para definir um texto como uma citação.

(início)
